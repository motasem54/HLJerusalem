# H.L. Jerusalem Stone & Marble Company

![H.L. Jerusalem Logo](logo.jpg)

## About The Project

Premium website for H.L. Jerusalem Stone and Marble Company - One of Palestine's leading natural stone and marble manufacturers since 1979.

## Features

### Frontend (English)
- Modern, luxury design with brown, black, and gold color scheme
- Responsive layout for all devices
- Homepage with company showcase
- Products catalog (Blocks, Slabs, Tiles, Cut to size)
- Projects gallery
- About company & international branches
- Contact page

### Admin Dashboard (Arabic)
- Marble/Stone categories management
- Products/Items management
- Image gallery management
- Projects management
- Company information management
- User authentication system

## Technology Stack

- **Backend:** PHP (Native)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript
- **Design:** Responsive, Modern UI

## Installation

1. Clone the repository:
```bash
git clone https://github.com/motasem54/HLJerusalem.git
```

2. Import database:
```bash
mysql -u root -p hljerusalem < database/schema.sql
```

3. Configure database connection:
- Edit `config/database.php`
- Update credentials

4. Setup admin user:
- Access: `/admin`
- Default credentials will be created on first run
